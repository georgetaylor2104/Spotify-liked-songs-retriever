# Spotify-liked-songs-retriever
Have all of your liked songs printed out to you.

# Instructions
Go to the Spotify for Developers website and make an app. Then use the export command in the terminal for client ID and client secret. Then you can run main.py in the terminal. It will open up Spotify in your browser and ask for permissions. After agreeing, it will return you to the app, which will print out all of your liked songs in numbered format.
